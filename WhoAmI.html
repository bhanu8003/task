What's interesting about this page?

Parallax scrolling! This special technique in computer graphics works on the idea of moving a background image slower than the foreground image to create an illusion of depth.

Parallax scrolling allows for the creation of dynamic, interactive user experience resulting in an enhanced visual appeal with the added advantage of displaying all content in one page therefore reducing navigation and clicks by the user to access content.

jQuery framework has been used for developing this web page because of my experience working with JavaScript and jQuery for developing web pages. This page has been made compatible on mobile devices as well by using CSS3 media-queries.
